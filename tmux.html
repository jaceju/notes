tmux 指令

tmux ls 列出主機上所有的 session
tmux a -t session-id 將背景的 session 叫回，a 代表 attach

tmux 快速鍵

Ctrl+b 激活控制台；此時以下按鍵生效
系統操作 ? 列出所有快捷鍵；按 q 返回
d 將目前的 session 丟到背景去；這樣可以暫時返回 shell 界面
D 選擇要脫離的 session ；在同時開啟了多個 session 時使用
Ctrl+z 掛起當前 session
r 強制重繪未脫離的 session
s 選擇並切換 session ；在同時開啟了多個 session 時使用
: 進入命令行模式；此時可以輸入支持的命令，例如 kill-server 可以關閉服務器
[ 進入複製模式；此時的操作與 vi/emacs 相同，按 q 或 esc 退出
~ 列出提示信息緩存；其中包含了之前 tmux 返回的各種提示信息
session 操作 c 創建新視窗
& 關閉當前視窗
數字鍵 切換至指定視窗
p 切換至上一個視窗
n 切換至下一個視窗
l 在前後兩個視窗間互相切換
w 通過視窗列表切換視窗
, 重命名當前視窗；這樣便於識別
. 修改當前視窗編號；相當於窗口重新排序
f 在所有視窗中尋找指定文字
面板操作 " 將當前面板平分為上下兩塊 (垂直分割)
% 將當前面板平分為左右兩塊 (水平分割)
x 關閉當前面板
! 將當前面板置於新窗口；即新建一個窗口，其中僅包含當前面板
Ctrl+方向鍵 以 1 個單元格為單位移動邊緣以調整當前面板大小
Alt+方向鍵 以 5 個單元格為單位移動邊緣以調整當前面板大小
space 重新調整面板配置；依次包括 even-horizontal 、 even-vertical 、 main-horizontal 、 main-vertical 、 tiled
q 顯示面板編號
o 在當前窗口中選擇下一面板
方向鍵 移動光標以選擇面板
{ 向前置換當前面板
} 向後置換當前面板
先按 [, 再按方向鍵或換頁鍵 捲動當前面板的 buffer
Alt+o 逆時針旋轉當前窗口的面板
Ctrl+o 順時針旋轉當前窗口的面板
